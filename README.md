## Descrição
Este projeto tem como objetivo automatizar a leitura e análise das datas de vencimento de tintas e outros produtos a partir de imagens, utilizando OCR e BigQuery. 

Os dados extraídos são armazenados em nosso banco de dados e utilizados para prever datas de venda e comparar com o vencimento dos produtos. 

Com base nessas informações, desenvolvemos um dashboard interativo com insights de vendas por franquia e recomendações sobre onde direcionar produtos próximos do vencimento, visando otimizar o giro de estoque e reduzir perdas.

---

## Tecnologias Utilizadas
| Tecnologia | Finalidade |
|-------------|-------------|
| *Python* | Automação, tratamento de dados e notificações |
| *OCR (Reconhecimento Óptico de Caracteres)* | Leitura automática das datas nas imagens |
| *BigQuery* | Armazenamento e processamento dos dados |
| *Looker Studio* | Visualização e geração de insights |
| *Prophet (modelo de previsão)* | Estimativa das datas de venda dos produtos |

---

## Funcionalidades
- Leitura automática das datas de vencimento a partir de imagens 
- Armazenamento estruturado dos dados no BigQuery 
- Modelo preditivo de vendas com o Prophet 
- Dashboard interativo com insights por franquia e recomendações de realocação 
- Script em Python para notificação automática de produtos próximos do vencimento 

---

## Resultados e Benefícios
- Redução de perdas por vencimento de produtos 
- Melhor distribuição de estoque entre as lojas 
- Tomada de decisão orientada a dados e previsões confiáveis
